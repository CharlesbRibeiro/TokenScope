# Dados locais

- `raw/`: dados originais e imutáveis;
- `interim/`: transformações intermediárias;
- `processed/`: dados limpos e consolidados;
- `samples/`: amostras pequenas e não sensíveis para testes.

Dados brutos são ignorados pelo Git, exceto `.gitkeep`. Não inclua segredos, dados pessoais ou conteúdo sem permissão. Preserve fonte, data, versão e run ID quando disponíveis.
