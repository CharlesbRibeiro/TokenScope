# Outputs locais

- `runs/`: artefatos de execuções identificáveis;
- `reports/`: relatórios gerados;
- `risk_passports/`: Risk Passports futuros.

O conteúdo gerado é local e ignorado pelo Git, exceto os marcadores `.gitkeep`. Não apague ou sobrescreva runs e evidências anteriores; crie novas versões.
